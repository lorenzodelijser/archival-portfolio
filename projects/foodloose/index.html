Assignment

This web development project was assigned by the academy, in the division: Tech. We were assigned to combine databases to develop a new feature. The project was created using HTML, CSS, and JavaScript.

For this project, we combined data from the Dunkin' Donuts and Spotify. When you visit the website, you can enter what you have consumed at Dunkin' Donuts. The website generates a playlist from data of Spotify and Dunkin' Donuts.

Process

A lot of ideas were gathered by making paper prototypes. I also made some little prototypes in HTML and CSS to explore possibilities.
